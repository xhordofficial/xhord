# XHORD Brand Website

A modern, immersive landing page for XHORD - a tech and creative agency that focuses on emotional storytelling and brand building.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- Tailwind CSS
- Three.js / React Three Fiber
- Framer Motion
- shadcn-ui

## Development

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

## Features

- Immersive 3D hero section with Three.js
- Smooth scroll animations
- Responsive design
- Modern UI components
- SEO optimized
